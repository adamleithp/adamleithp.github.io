From default to hero.

A developers personal checklist to setting up a new Macbook Pro 16" from scratch; changing system configurations from default settings.

Uncluttered, minimal macosx experience that let's you focus on your work. Here's the checklist.

  1. Security

    • Filevault On - Important

      System Preferences → Secuity → Filevault → Turn on Filevault.

    • Firewall On - Important

      System Preferences → Secuity → Firewall → Turn on Firewall.

  2. Keyboard

    • Hide Language Flag in status bar - Minimalist

      System Preferences → Keyboard → Input Sources → Uncheck "Show input menu in menu bar".

  3. General Appearance

    • Dark mode, always on - improve UX

      System Preferences → General → Appearances "Dark"

    • Hide Menu Bar (top bar) automatically - Minimalist / improve UX

      System Preferences → General → Check "Automatically hide and show the menu bar"

  4. Screensaver

    • Hot corner bottom right show desktop - improve UX

      System Preferences → Desktop Screensaver → Hot Corners → Bottom right, set to "Show Desktop"

  5. Display

    • Turn off auto brightness on battery - improve UX

      System Preferences → Energy Saver → Uncheck "Slightly dim the disaply while on battery power"

  6. Dock

    • Hide Dock - maximize screen real-estate

      System Preferences → Dock → Check "Automatically hide and show the dock"

  7. Touch Bar

    • Show expanded/full touchbar settings by default - impove UX

      System Preferences → Keyboard → Keyboard Tab → Touchbar shows "Expanded Control Strip"

  8. Screenshots

    • Set folder to save screenshots in a particular folder location. (example: Documents/Screenshots) - Minimalist

      defaults write com.apple.screencapture location /Users/adamleithp/Documents/Screenshots

    • Turn off floating thumbnails (those annoying thumbnails that hang around for 5 seconds - Minimalist / improve UX

      Screenshot.app → Options → Uncheck Show Floating Thumbnail
